10.Checkbox

  • [CheckBox] 특정 항목 표시하지 않기
  • [CheckBox] 특정 항목 비활성화 하기
  • [CheckBox] DataList와 연결하여 선택 항목을 설정하기
  • [CheckBox] 특정 선택 항목들을 함수를 사용하여 동적으로 비활성화하기
  • [CheckBox] 선택 항목을 직접 입력하고, 선택 항목을 표시하는 방법을 설정하기
  • [CheckBox] CheckBox의 각 선택 항목에 label 값 혹은 사용자 지정값을 툴팁으로 표시하기
  • [CheckBox]선택된 항목 값 가져오기
  • [Checkbox]선택된 항목 행(Row)의 수 조정하기
  • [CheckBox] 선택 항목 열(cols)의 수 조정하기
  • [CheckBox] 선택 항목 전체 선택 / 해제 하기
  • [CheckBox] Event - onchange (값이 변경될 때 발생)
  • [CheckBox] Event - onlabelClick (label이 클릭될 때 발생)
  • [Checkbox] 컴포넌트의 값(value)과 데이터 객체(DataMap, DataList) 연결하기
