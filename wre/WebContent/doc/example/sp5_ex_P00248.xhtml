[Radio] 표시 항목의 열과 행 바꾸기

1개요

Radio컴포넌트의 표시 항목의 열과 행을 변경하는 예제입니다.

이 기능은 아래의 속성으로 사용할 수 있습니다.
- cols : 선택 항목을 표시할 열의 수
- rows : 선택 항목을 표시할 행의 수

2구현된 기능

  • 열 바꾸기

  • 행 바꾸기

3예제 테스트 방법

3.1열 바꾸기

STEP1. 실행 결과를 확인합니다.

속성 cols = "3" 이 설정되어서 Radio컴포넌트를 표시할 때에 3열로 표시하는 것을 확인합니다.

[브라우저(Chrome) 실행 예시]

3.2행 바꾸기

STEP1. 실행 결과를 확인합니다.

속성 rows = "3" 이 설정되어서 Radio컴포넌트를 표시할 때에 3행으로 표시하는 것을 확인합니다.

[브라우저(Chrome) 실행 예시]

4구현 예시

4.1Radio컴포넌트의 열과 행 바꾸기

Radio의 속성을 정의합니다.

  • [필수] cols //선택 항목을 표시할 열의 수. "2"로 정의할 경우, 2열에 걸쳐 선택 항목이 표시됨

  • [필수] rows //선택 항목을 표시할 행의 수. "2"로 정의할 경우, 2행에 걸쳐 선택 항목이 표시됨.

그림 1.웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시

<!-- radio 의 소스 본문 예시 -->
<xf:select1 cols="3" id="rad_exam1">
    <xf:choices>
        <xf:itemset nodeset="data:dlt_dataList1">
            <xf:label ref="City"></xf:label>
            <xf:value ref="Code"></xf:value>
        </xf:itemset>
    </xf:choices>
</xf:select1>

그림 2.웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시

<!-- radio 의 소스 본문 예시 -->
<xf:select1 rows="3" id="rad_exam2">
    <xf:choices>
        <xf:itemset nodeset="data:dlt_dataList1">
            <xf:label ref="City"></xf:label>
            <xf:value ref="Code"></xf:value>
        </xf:itemset>
    </xf:choices>
</xf:select1>
