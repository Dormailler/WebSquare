[Radio] 특정 선택 항목들을 선택 항목 목록에 표시하지 않고 숨기기

1개요

Radio컴포넌트에서 DataList와 연결하여 선택 항목을 표시할 경우,
연결된 DataList의 특정 컬럼 값을 기준으로 일부 선택 항목을 숨기는 예제입니다.

이 기능은 아래의 속성과 함수로 사용할 수 있습니다.
- visibleColumn : (속성)선택 항목 목록에서 특정 항목을 감추는 기능
- visibleColumnFalseValue : (속성)visibleColumn 사용 시 목록에서 특정 항목을 감추기 위한 조건에 해당하는 값
- setVisibleColumn : (함수)visibleColumn 속성을 동적으로 설정

4구현 예시

4.1특정 항목 감추기 - 속성

STEP1. Radio의 속성을 지정합니다.

  • [필수] visibleColumn //선택 항목 목록에서 특정 항목을 감추는 기능. 선택 항목 목록이 DataList와 바인딩된 경우, 표시 여부를 알려주는 컬럼을 지정. 표시 여부를 알려주는 컬럼의 값이 visibleColumnFalseValue 속성에 지정한 값과 같은 경우, 해당 행은 선택 항목 목록에 표시되지 않음. selectedIndex는 영향을 받지 않음. 즉 목록에 보이는 인덱스가 아닌 전체 인덱스를 기준으로 동작. 해당 속성은 setVisibleColumn 함수를 통해 동적으로 설정 가능.

  • [필수] visibleColumnFalseValue //[default: "0", "false", "FALSE", 혹은 "F"] visibleColumn 사용 시 목록에서 특정 항목을 감추기 위한 조건에 해당하는 값

그림 1.웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시

[소스 코드 예시]

<xf:select1 visibleColumn="Type" visibleColumnFalseValue="Asia" id="rad_exam2">
    <xf:choices>
        <xf:itemset nodeset="data:dlt_dataList1">
            <xf:label ref="City"></xf:label>
            <xf:value ref="Code"></xf:value>
        </xf:itemset>
    </xf:choices>
</xf:select1>

4.2특정 항목 감추기 - 함수

STEP1. Radio의 함수를 정의합니다.

Radio컴포넌트와 연결된 DataList의 특정 컬럼 값을 기준으로 일부 선택 항목을 숨기기 위해 아래의 함수를 사용합니다.

  • [선택] setVisibleColumn // visibleColumn 속성을 동적으로 설정. visibleColumn 속성으로 설정된 컬럼은 Radio와 바인딩된 DataList의 컬럼 중 선택 항목으로의 표시 여부에 대한 설정 값을 가지고 있음. visibleColumn 속성은 Radio의 선택 항목 목록 중 특정 항목을 감추기 위한 용도로 사용됨. selectedIndex는 영향을 받지 않고 항상 전체 목록을 기준으로 동작.

STEP2. setVisibleColumn를 사용하여 유효성 검사를 실행합니다.

[소스 코드 예시]

/**
 * 버튼 [ 특정 항목 감추기 ] 클릭 시
 */
scwin.btn_ex1_onclick = function (e) {
    // Radio 'rad_exam3'와 연결되어 있는 DataList 'dlt_dataList1'의 Type이 Europe인 데이터는 제외하고 표시하기
    rad_exam3.setVisibleColumn("Type", "Europe");
};

5주요 API
