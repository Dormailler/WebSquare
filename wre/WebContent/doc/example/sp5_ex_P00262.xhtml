[CheckBox] 특정 선택 항목들을 함수를 사용하여 동적으로 비활성화하기

1개요

CheckBox컴포넌트의 특정 선택항목들을 함수를 사용하여 동적으로 비활성화하는 예제입니다.

이 기능은 아래의 함수로 사용할 수 있습니다.
- setEnableColumn : (함수)enableColumn 속성을 동적으로 설정.

P00262

2구현된 기능

  • 선택항목 비활성화하기

버튼 비활성화하기를 클릭합니다.

STEP2. 실행 결과를 확인합니다.

4구현 예시

4.1선택항목 비활성화하기

STEP1. CheckBox의 함수를 정의합니다.

CheckBox컴포넌트에 선택 항목들을 함수를 사용하여 동적으로 비활성화하기 위해 아래의 함수를 사용합니다.

  • [선택] setEnableColumn // (함수)enableColumn 속성을 동적으로 설정.

    enableColumn 속성으로 설정된 컬럼은 Checkbox와 바인딩된 DataList의 컬럼 중 선택 항목으로의 disable 여부에 대한 설정 값을 가지고 있음.

    enableColumn 속성은 Checkbox의 선택 항목 목록 중 특정 항목을 disable 시키기 위한 용도로 사용됨. selectedIndex는 영향을 받지 않고 항상 전체 목록을 기준으로 동작.

STEP2. setEnableColumn을 사용하여 CheckBox컴포넌트의 특정 선택 항목들을 비활성화합니다.

[소스 코드 예시]

/**
 * 버튼 [비활성화하기] 클릭 시
 */
scwin.btn_ex1_onclick = function (e) {
    // CheckBox 'cbx_exam1'에 DataList 'dataList1'을 연결합니다.
    cbx_exam1.setEnableColumn("Country", "Korea");
};
