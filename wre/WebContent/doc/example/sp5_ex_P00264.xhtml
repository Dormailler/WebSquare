[CheckBox] 선택 항목을 직접 입력하고, 선택 항목을 표시하는 방법을 설정하기

1개요

CheckBox컴포넌트의 선택 항목을 직접 입력하고, 선택 항목을 표시하는 방법을 설정하는 예제입니다.

이 기능은 아래의 속성으로 사용할 수 있습니다.
- cols : (속성)선택 항목을 표시할 열의 수.
- rows: (속성)선택 항목을 표시할 행의 수.

P00264

2구현된 기능

  • 직접입력한 데이터 표시하기

  • 선택 항목 3열로 표시하기

  • 선택 항목 3행으로 표시하기

3예제 테스트 방법

3.1직접입력한 데이터 표시하기

STEP1. 실행 결과를 확인합니다.

직접 입력한 DataList가 아래 그림과 같이 선택 항목으로 표시되는 것을 확인합니다.

※직접 데이터를 입력하는 법은 4.1 참고

[브라우저(Chrome) 실행 예시]

3.2선택 항목 3열로 표시하기

STEP1. 실행 결과를 확인합니다.

직접 입력한 DataList가 아래 그림과 같이 3열로 표시되는 것을 확인합니다.

※데이터의 열을 입력하는 법은 4.2 참고

[브라우저(Chrome) 실행 예시]

3.3선택 항목 3행으로 표시하기

STEP1. 실행 결과를 확인합니다.

직접 입력한 DataList가 아래 그림과 같이 3행으로 표시되는 것을 확인합니다.

※데이터의 행을 입력하는 법은 4.3 참고

[브라우저(Chrome) 실행 예시]

4구현 예시

4.1직접입력한 데이터 표시하기

STEP1. 더블 클릭하기

WebSquare 스튜디오에서 DataList와 연결하고 싶은 CheckBox컴포넌트를 더블 클릭합니다.

[브라우저(Chrome) 실행 예시]

STEP2. Item 추가하기

'CheckBox 설정' 팝업 창에서 추가하고자 하는 데이터의 컬럼 수 만큼 'Item 추가' 버튼을 클릭합니다.

[브라우저(Chrome) 실행 예시]

STEP3. Label, Value 설정하기

데이터의 Label, Value값을 설정 후 Span Direction을 'none'으로 설정 후 'OK' 버튼을 누릅니다.

[브라우저(Chrome) 실행 예시]

STEP4. 실행 결과를 확인합니다.

아래 그림과 같이 STEP3에서 수작업으로 입력한 데이터가 표시되고 있는 것을 확인합니다.

[브라우저(Chrome) 실행 예시]

4.2선택 항목 3열로 표시하기

STEP1. Span Direction, Span Count 설정하기

4.1의 STEP3에서 Span Direction을 cols로, Span Count를 3으로 설정 후 'OK' 버튼을 누릅니다.

[브라우저(Chrome) 실행 예시]

4.3선택 항목 3행으로 표시하기

STEP1. Span Direction, Span Count 설정하기

4.1의 STEP3에서 Span Direction을 rows로, Span Count를 3으로 설정 후 'OK' 버튼을 누릅니다.

[브라우저(Chrome) 실행 예시]
