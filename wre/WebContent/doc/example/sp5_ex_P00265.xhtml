[CheckBox] CheckBox의 각 선택 항목에 label 값 혹은 사용자 지정값을 툴팁으로 표시하기

1개요

CheckBox컴포넌트의 각 선택 항목에 label 값 혹은 사용자 지정값을 툴팁으로 표시하는 방법을 설정하는 예제입니다.

이 기능은 아래의 속성으로 사용할 수 있습니다.
- tooltipDisplay : (속성)사용자가 선택한 값에 대한 툴팁 표시 여부.
- tooltipShowAlways: (속성)텍스트 길이에 상관 없이 마우스 오버시 툴팁을 항상 표시할 것인지의 여부.
- tooltipFormatter: (속성)tooltipDisplay="true"인 경우, 툴팁 내용을 변경할 사용자 함수명.

4구현 예시

4.1라벨이 짤려있는 선택 항목 툴팁 표시하기

CheckBox의 속성을 정의합니다.

  • [필수] tooltipDisplay //[default: false, true] 사용자가 선택한 값에 대한 툴팁 표시 여부.

그림 1.웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시

<!-- selectBox 의 소스 본문 예시 -->
<xf:select tooltipDisplay="true" id="cbx_exam2">
    <xf:choices>
        <xf:itemset nodeset="data:dataList11">
            <xf:label ref="city"></xf:label>
            <xf:value ref="country"></xf:value>
        </xf:itemset>
    </xf:choices>
</xf:select>

4.2모든 선택 항목 툴팁 표시하기

CheckBox의 속성을 정의합니다.

4.3사용자 함수를 사용하여 툴팁 표시하기

STEP1. CheckBox의 속성을 정의합니다.

  • [필수] tooltipShowAlways //tooltipDisplay="true"인 경우, 툴팁 내용을 변경할 사용자 함수명. 사용자 함수는 label 및 index를 인자로 받아 툴팁으로 사용할 스트링을 반환.

그림 3.웹스퀘어5 SP5 스튜디오의 Property View(속성창) 예시

[소스 코드 예시]

<xf:select tooltipDisplay="true" tooltipShowAlways="true" tooltipFormatter="scwin.tooltip" id="cbx_exam4">
    <xf:choices>
        <xf:itemset nodeset="data:dataList11">
            <xf:label ref="city"></xf:label>
            <xf:value ref="country"></xf:value>
        </xf:itemset>
    </xf:choices>
</xf:select>
STEP2. 사용자 지정 함수 scwin.tooltip를 정의 합니다.

[소스 코드 예시]

/**
 * tooltipFormatter속성에 설정할 사용자 정의 함수
 */
scwin.tooltip = function (label, index) {
    let data = dataList1.getDataRow(index);
    let value = data.country;
    // 커서가 오버되어 있는 항목에 라벨 + 나라명으로 툴팁을 표시합니다.
    return label + ", " + value;
};
